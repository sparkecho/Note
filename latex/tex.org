1. tlmgr - the TeX Live Manager

2. $ tlmgr search xecjk
   tlmgr cannot setup TLPDB in /root/texmf at /usr/bin/tlmgr line 5435.

   解决方法：
   $ tlmgr init-usertree
   会创建 texmf 目录, 用来存放安装的包
   
   如果报出如下错误：
   /usr/bin/tlmgr: could not find a usable xzdec.
   则, $ sudo apt-get install xzdec
