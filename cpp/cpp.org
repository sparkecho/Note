C++ Note

* 1. cpp/c 中的兼容性问题
  在 gcc 默认一个未初始化的指针为 NULL, VC++6.0 中并不默认为 NULL, 所以有可能会导致不兼容
  解决方法: 比如说构建链表, 一个节点的指针域最好都初始化为 NULL.
   

* 2. 判断输入的内容是否符合预期
  当 cin.fail() == true 时, 说明输入的内容不符合预期
  e.g. 
      int e;
      cin >> e;
        
      如果输入的值的类型为 int 类型, 那么 cin.fail() == false;
      否则 cin.fail() == true.
