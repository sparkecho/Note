* 中文编码
创建table的时候就使用utf8编码

举个例子：
复制代码

create table entries2 (
        id     int auto_increment, 
        title  text,
        content  text,
        posted_on  datetime,
        primary key (id)   
) character set = utf8;

复制代码

在每次创建表的时候都在最后加上 

　　character set = utf8

就可以很好的支持中文。
